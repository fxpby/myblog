# SSE MCP Server playground

## 1. 功能目标

- 实现一个 SSE 传输的 MCP 服务器
- 使用 Client 如 `CheeryStudio` 借助 LLM 调用本地运行的 MCP 服务器抓取分析数据

## 2. 环境准备

- 安装 node.js， 版本 18+
- 安装 pnpm 包管理器
- 安装客户端，自己用的`CheeryStudio`
- 准备个大模型的 api key
