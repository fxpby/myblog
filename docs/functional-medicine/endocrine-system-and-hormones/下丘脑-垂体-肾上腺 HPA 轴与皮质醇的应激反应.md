# 下丘脑-垂体-肾上腺 HPA 轴与皮质醇的应激反应

## 肾上腺内分泌

肾上腺皮质激素属于类固醇激素，又称为甾体激素，按照生理作用分为三种：

### 糖皮质激素

17-羟肾上腺皮质醇，简称皮质醇

商品名是氢化可的松

主要作用是调节糖、蛋白质和脂肪的代谢，从而影响全身，尤其是肝、肌肉、心、脑等重要脏器

### 盐皮质激素

以醛固酮为代表，主要作用于钾、钠、氯及水代谢。

### 性激素类肾上腺皮质激素

肾上腺皮质分泌的性激素主要是一些弱的雄激素，以脱氢表雄酮为代表，其中极小部分在肾上腺可转化成活性很强的雄激素睾酮，也可转化成部分雌激素。

## 肾上腺皮质激素的合成、转运及降解

### 生物合成

胆固醇是主要前体，经一系列酶促反应在不同皮质带（球状带、束状带、网状带）分别生成盐皮质激素（醛固酮）、糖皮质激素（皮质醇）和性激素（DHEA、睾酮）。

### 转运

皮质激素主要通过血浆蛋白结合形式运输，游离部分具有生物活性（如皮质醇仅 1%~3%游离）。

### 降解与排泄

肝脏是主要降解场所，通过还原、结合（如葡糖醛酸化）等反应转化为水溶性代谢物，最终由肾脏排出。

## 下丘脑-垂体-肾上腺轴 HPA 轴

### HPA 轴层级调控

下丘脑（CRH）→ 垂体（ACTH）→ 肾上腺（皮质醇、DHEA）形成负反馈循环。

### 核心激素功能

- CRH：促肾上腺皮质激素释放激素，刺激 ACTH 分泌，并影响交感神经活性。
- ACTH：促肾上腺皮质激素，促进皮质醇合成，维持肾上腺皮质结构，具有昼夜节律和应激反应。
- GC：皮质醇，调节糖、脂肪、蛋白质代谢，影响应激反应，长期过量导致库欣综合征。
- DHEA：脱氢表雄酮，作为性激素前体，随年龄下降，拮抗皮质醇部分负面效应。

### 病理与临床意义

- 外源性 GC 长期使用可抑制 HPA 轴，突然停药引发肾上腺危象。
- HPA 轴失衡与代谢异常、免疫失调及衰老相关疾病密切相关。

## 应激反应

### 应激反应的分类

#### 1. 生理性应激（良性应激）

- 对**轻度或短暂的内外刺激**作出的反应。
- 有助于调动潜能，不显著影响机体功能，甚至有助于**战斗或逃跑**行为。
- 例如：体育竞赛、饥饿、考试等。

#### 2. 病理性应激（恶性应激）

- 因**应激原强烈且持续**，超出了机体适应能力，导致内环境失衡。
- 可引起**功能紊乱、代谢异常、组织损伤甚至死亡**。
- 常见于：长期工作压力、慢性心理打击、严重疾病等。

#### 3. 按来源分类

- **心理应激**：如丧亲、恐惧、焦虑、愤怒等情绪。
- **生理应激**：如疼痛、外伤、感染、低血糖等。
- **环境应激**：如高温、低温、噪声、污染、光线等。
- **营养应激**：如饥饿、营养不均衡。
- **氧化应激**：如自由基异常活跃引起细胞损伤。
- **肾上腺应激**：与肾上腺系统的异常反应有关。

### 应激反应的神经内分泌机制

> 应激反应主要通过**蓝斑-交感-肾上腺髓质系统**和**下丘脑-垂体-肾上腺皮质轴（HPA 轴）**进行调节。

#### 1. 蓝斑-交感-肾上腺髓质系统

- 是机体对**急性应激**反应的**快速系统**。
- 激活后会导致：

  - **中枢效应**：引起兴奋、警觉、紧张、焦虑等情绪。
  - **外周效应**：
    - 心血管系统：心率增快、心肌收缩力增强，血液重新分配。
    - 呼吸系统：支气管扩张，增加通气量。
    - 代谢系统：糖原分解、脂肪动员、血糖升高，能量供给增加。
    - 促进多种激素（如 ACTH、甲状腺激素等）的分泌。

- **可能的不利影响**：
  - 腹腔器官缺血
  - 能量物质大量消耗
  - 心血管应激性损伤（如心肌缺血、血栓、心律失常）
  - 血压升高

#### 2. 下丘脑-垂体-肾上腺轴（HPA 轴）

- 是应激反应中的**慢速、长效调节系统**。
- 应激时，**下丘脑释放 CRH**，促使垂体分泌 ACTH，进而导致肾上腺皮质分泌**皮质醇**。
- 皮质醇是**重要的压力激素**，具有以下作用：

  - 促进糖异生、升高血糖
  - 抑制胰岛素作用，维持高血糖状态
  - 稳定细胞膜，减少炎症介质的释放
  - 具有抗炎、抗过敏、抗感染作用
  - 调节免疫反应、影响性激素分泌、甲状腺功能等

- **不利影响**：
  - 持续高水平皮质醇可抑制免疫功能，使机体易感染
  - 导致性功能减退、月经失调、泌乳减少
  - 抑制甲状腺功能与生长激素分泌
  - 引发抑郁、食欲减退、自杀倾向等心理问题

#### 其他激素变化

- **升高的激素**：β-内啡肽、抗利尿激素（ADH）、醛固酮、胰高血糖素、催乳素
- **降低的激素**：胰岛素、TRH、TSH、GH、LH、FSH
- 例如：生长激素在**急性应激**时增加，但**慢性应激**时会减少

### 应激引起的系统功能变化

#### 1. 代谢变化

- **特点**：分解代谢增强、合成代谢减弱，代谢率升高
- **表现**：
  - 糖原分解增强 → 血糖升高（可出现**应激性糖尿**）
  - 脂肪分解增加 → 血液中游离脂肪酸和酮体增加
  - 蛋白质分解增强 → 出现**负氮平衡**
- **不利影响**：能源消耗大、消瘦、免疫力下降、伤口愈合缓慢等

#### 2. 心血管系统

- **变化**：心率增快、心肌收缩力增强、外周阻力升高
- **影响**：
  - 保证心、脑等重要器官供血
  - 但长期应激可导致心肌缺血、心律失常、甚至猝死

#### 3. 消化系统

- **变化**：食欲减退、胃肠黏膜受损、肠道蠕动异常
- **表现**：
  - 胃肠血管收缩 → **应激性溃疡**
  - 结肠蠕动加快 → 营养吸收不良
- **长期影响**：可能导致肠易激综合征、克罗恩病、肠漏综合征等

#### 4. 免疫系统

- **急性应激**：非特异性免疫增强，如中性粒细胞增多、吞噬活性增强
- **慢性应激**：免疫功能被抑制，易感染、自身免疫疾病发生率升高

#### 5. 血液系统

- **急性应激**：血液凝固性增强、血小板聚集性升高、红细胞沉降率加快
- **慢性应激**：可能出现**低血色素性贫血**，补铁无效

#### 6. 泌尿生殖系统

- **急性应激**：尿量减少、尿比重升高、尿钠降低
- **机制**：交感神经兴奋、肾素-血管紧张素系统激活、醛固酮与抗利尿激素分泌增加
- **持续应激**：可能导致肾小管坏死

## THA 轴功能失衡的干预方案

### HPA 轴功能异常的发展过程

- **HPA 轴功能异常**的整个过程是一个**由失衡到衰退**的**连续性变化**，而不是一个明确的“时间点”。
- 从 HPA 轴活性升高到活性减退，是一个**渐进过程**，因此功能医学从**整体出发**来评估和干预。

### 应激源的识别与分析

#### 主要应激源

1. **压力**（生理或心理）
2. **持续性创伤或创伤后状态**
3. **慢性炎症/感染**
4. **营养缺乏**
5. **低血糖**
6. **食物过敏或食物不耐受**
7. **菌群失调/肠漏综合征**
8. **毒素过量**

### 干预策略

#### 1. 调整生活方式

- **识别并消除**对个体健康状态有影响的生活方式因素。
- 个性化评估，调整日常行为模式以降低应激水平。

#### 2. 诱导放松与情绪调整

- **放松疗法**是调节 HPA 轴活性的重要方法。
- 方法包括：
  - 生物反馈
  - 自主神经训练
  - 瑜伽
  - 冥想
  - 能量医学
- 放松可**减轻交感神经激活**，降低如心率加快、外周血管收缩及高血压等指标。
- 交感神经受 CRH（促肾上腺皮质激素释放激素）调控，放松可**降低 CRH 活性**，进一步降低皮质醇水平，改善生理与心理状态。

#### 3. 适度锻炼

- 增加细胞对激素（如皮质醇、胰岛素、甲状腺激素）的**敏感性**。
- 帮助机体**适应压力**，提高应激反应的**调节能力**。

#### 4. 调整饮食

- **推荐饮食策略**：
  - **排除饮食法**（Elimination Diet）：排除潜在致敏或不耐受食物，减少炎症反应。
  - **低糖饮食**：降低血糖负荷，每餐建议摄入非精制碳水化合物、高质量蛋白质和不饱和脂肪、丰富的植物化合物。
  - **规律饮食**：少食多餐，避免过量进食时间对节律造成干扰。
  - **避免刺激性食物**：如咖啡因、精制糖、巧克力等，这些可能加剧应激反应。

### 肾上腺基础的营养支持

#### 1. 复合维生素 B

- **作用**：激素合成的辅因子，调节神经递质释放。
- **推荐剂量**：
  - 维生素 B5（泛酸）：1000–1500 mg/天
  - 维生素 B6（吡哆醇）：50–100 mg/天
  - 生物素：1000 mcg/天
  - 叶酸：400–800 mcg/天
- **目的**：补充长期压力导致的 B 族维生素流失。

#### 2. 维生素 C 和抗氧化复合配方

- **作用**：帮助清除自由基，减少应激引起的氧化损伤。
- **推荐剂量**：1–2 g 维生素 C/天，配合抗氧化剂使用。

#### 3. 镁

- **作用**：调节神经兴奋性，减少应激反应。
- **注意**：低镁和高钙会加剧应激激素的释放，影响镁的再吸收。
- **推荐剂量**：400–600 mg/天

#### 4. 磷脂酰丝氨酸（Phosphatidylserine, PS）

- **作用**：
  - 能够穿透血脑屏障，直接影响大脑功能。
  - 降低紧张状态下皮质醇水平。
  - 提高大脑中影响情绪的神经递质（如乙酰胆碱、多巴胺）水平。
- **推荐剂量**：600–800 mg/天
- **使用建议**：若晚上皮质醇偏高，可在**睡前**服用。

### 植物化合物营养素

#### 1. 治疗肾上腺活性亢进的植物营养素

- **红景天**（Rhodiola rosea）

  - 作用：抗疲劳、改善注意力、降低觉醒时皮质醇反应。
  - 推荐配方：3% 肉桂醇甙和 1% 红景天苷
  - 推荐剂量：200–600 mg/天

- **金丝桃**（Hypericum perforatum）
  - 作用：缓解兴奋、疲劳和抑郁。
  - 适用于 HPA 轴过度激活的患者（高 ACTH / 高皮质醇）。
  - 推荐剂量：金丝桃提取物（类黄酮成分）200–600 mg/天
  - **效果**：两周后显著降低血浆 ACTH 和皮质酮水平。

#### 2. 治疗肾上腺功能减退的植物营养素

- **人参**（Panax ginseng）

  - 作用：改善焦虑，调节 HPA 轴反馈循环，提高肾上腺功能。
  - 适合长期应激导致肾上腺疲劳的患者。
  - 推荐剂量：100–200 mg，每日两次
  - **效果**：在实验中能够明显提升体力、改善皮质素水平。

- **甘草**（Glycyrrhiza glabra）
  - 作用：抑制皮质醇转化为主（皮质酮），从而**延长皮质醇在体内的停留时间**。
  - 推荐配方：25% 标准化甘草酸制剂
  - 推荐剂量：600–800 mg/天
  - **注意**：甘草有升压作用，**不适合高血压患者**。

#### 3. 激素疗法

- **适用情况**：当患者出现低皮质醇和/或低 DHEA（脱氢表雄酮）时。
- **药物**：DHEA（脱氢表雄酮）补充剂。
- **推荐剂量**：
  - 女性：5–25 mg，每日两次
  - 男性：10–50 mg，每日两次
- **效果**：
  - 提高血清雄激素水平至正常范围
  - 改善性功能、情绪、自信、疲劳
- **注意事项**：
  - 必须在**功能医学专业医生**指导下使用
  - 需先检测 **DHEA 和 DHEA-S 水平**，并监测其他激素（如睾酮、雌二醇）
  - 副作用：
    - 女性可能出现痤疮、多毛
    - 男性可能表现出好斗、易怒
    - 对雄激素过多的女性应**谨慎使用**

### 干预方案总结

| **干预方向**     | **方法**                      | **作用**                                                         |
| ---------------- | ----------------------------- | ---------------------------------------------------------------- |
| **应激源识别**   | 排除饮食、检测慢性炎症等      | 从根源控制应激反应                                               |
| **生活方式调整** | 改善作息、规律饮食、适度运动  | 提高机体适应性，减少外源性压力                                   |
| **情绪调节**     | 放松疗法、冥想、情绪管理      | 降低交感神经激活，减少 HPA 轴兴奋性及皮质醇分泌                  |
| **营养支持**     | 维生素 B、C、镁、PS、红景天等 | 提供神经、内分泌系统所需的营养基质，减轻应激造成的代谢与生理负担 |
| **植物营养素**   | 金丝桃、人参、甘草            | 调节 HPA 轴活性，改善肾上腺功能异常                              |
| **激素疗法**     | DHEA 补充剂                   | 在严重肾上腺功能不足时提供外源性支持，需专业检测和规范使用       |
