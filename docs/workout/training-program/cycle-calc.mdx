## 计算思路

### 1. 准备数据

- 1RM 重量
- 第一周初始缓冲区
- 渐进超负荷增强比率
- 减载比率
- 轻训比率
- 1RM-重复次数的关系表

  | %1RM | 能够重复次数 |
  | ---- | ------ |
  | 100  | 1      |
  | 95   | 2      |
  | 93   | 3      |
  | 90   | 4      |
  | 87   | 5      |
  | 85   | 6      |
  | 83   | 7      |
  | 80   | 8      |
  | 77   | 9      |
  | 75   | 10     |
  | 70   | 11     |
  | 67   | 12     |
  | 65   | 15     |

### 2. 周

根据自己的训练计划安排对应的渐进超负荷周和减载周，如 3 周渐进超负荷，1 周减载，减载周可以分前半段和后半段或者很多段根据自己的分化安排

### 3. 组数&次数

根据周期计划自行安排

### 4. 绝对强度

- 重量：1RM * 绝对强度比率
- 绝对强度比率：根据`1RM-重复次数的关系表`计算

### 5. 相对强度

第一周的绝对强度比率为`（ 1 - 第一周初始缓冲区）`，即假设`缓冲区`是 15%，那么第一周相对强度就是 85%。其他渐进超负荷周按照`渐进超负荷增强比率`递增，减载周按照`减载比率`递减。第二个中周期的第一个渐进超负荷周相对强度是在**第一个中周期最后一个渐进超负荷周**基础上按照`渐进超负荷增强比率`递增

### 6. 训练负荷

- 重量：绝对强度重量 * 相对强度比率
- 比率：绝对强度比率 * 相对强度比率

### 7. 容量

组数 `*` 次数 `*` 训练负荷重量

### 8. 轻训

训练负荷重量 * 轻训比率
