## ECMAScript 内置对象

Boolean、Number、String、RegExp、Date、Error

```ts
let a: Boolean = new Boolean(1)
let b: String = new String('olu')
let c: Number = new Number(1)
let d: Date = new Date()
let e: RegExp = /^1/
let f: Error = new Error('error~')
```
